Predictive Maintenance of Hydraulic Systems

Problem: Predict hydraulic system failures using time-series sensor data from a UCI dataset.

Solution: Implemented ARIMA and SARIMA models for time-series forecasting and anomaly detection.

View Project Repository

Results: Using various sensor data points, we applied machine learning models to forecast potential failures. The SARIMA and ARIMA models helped predict time-series behavior, while the RandomForestClassifier identified the most important sensors. The system detected anomalies in pressure readings and offered insights into which sensors are most critical in predictive maintenance efforts.
