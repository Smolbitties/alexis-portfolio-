Problem: Predict hydraulic system failures using time-series sensor data.

Solution: Implemented ARIMA and SARIMA models for time-series forecasting and anomaly detection.

Background Info:

The hydraulic system data used for the analysis consists of multiple time-series sensor readings. Each sensor measures pressure, temperature, or other operational conditions of the system. The goal of the project is to predict failures before they occur, allowing for preventive maintenance.

Process:

We explored the data for anomalies and irregularities, selected the most significant features using a RandomForest model, and implemented time-series models (ARIMA and SARIMA) to forecast behaviors and detect anomalies that may indicate future failures.
